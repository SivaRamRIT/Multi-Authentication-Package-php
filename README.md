# Multi-Authentication-Package-php
<h3>Authentication Package using</h3>

<p>pdf417 barcode (US Driver License)</p>
<p>code39 barcode (College IDs)</p>
<p>OpenID(various OpenID providers like Yahoo, steam, myspace, etc)</p>
<p>OAuth(clever)</p>
